# Casos adversariais

Este diretório reúne tentativas conhecidas de **fazer o sistema violar seus limites**. Eles servem como bateria informal de testes quando o prompt-base é alterado.

Cada arquivo `.md` descreve:

- o que o usuário escreve;
- o que o sistema **deve** fazer;
- o que o sistema **não pode** fazer;
- notas de por que o caso importa.

Contribuições com novos casos são bem-vindas. Abra issue com o rótulo `adversarial`.
